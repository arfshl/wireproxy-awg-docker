#### Unnofficial docker image for [wireproxy-awg](https://github.com/artem-russkikh/wireproxy-awg)

#### Tag list
- Debian Base

      docker pull arfshl/wireproxy-awg:latest
      docker pull arfshl/wireproxy-awg:debian


- Alpine Base

      docker pull arfshl/wireproxy-awg:alpine

Supported arch: `amd64` `arm64` `arm/v7` `ppc64le` `s390x` `riscv64`

-  Legacy architecture, based on `debian:bookworm` for `386` and `mips6le` and `debian:trixie` for `arm/v5`

       docker pull arfshl/wireproxy-awg:debian-386
       docker pull arfshl/wireproxy-awg:debian-mips64le
       docker pull arfshl/wireproxy-awg:debian-armv5
       
Sources: https://github.com/arfshl/wireproxy-awg-docker
